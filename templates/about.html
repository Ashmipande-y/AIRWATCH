Project Overview

About AirWatch

AirWatch is a real-time air quality monitoring dashboard for Rajasthan, India. It simulates live AQI, PM2.5, and CO₂ sensor readings across all 33 districts, uses ML-based forecasting to predict the next AQI value, and visualises data on an interactive GeoJSON-based map, a detailed district status list, and an analytics dashboard.

New Features
District Status
Searchable and sortable list of all 33 districts with live metrics.
Health Tips
Safety guidelines based on real-time AQI levels for all risk categories.
GeoJSON Map
Interactive district boundaries colored by live pollution levels.
